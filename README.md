# Analiza konwersji w branży automotive

## O projekcie
Analiza lejka sprzedażowego dla salonu samochodowego.
Celem projektu było sprawdzenie, jak skutecznie reklamy przekładają się na realną sprzedaż samochodów.

## Użyte technologie
* **SQL** (SQLite)
* **Techniki:** JOIN, agregacja (GROUP BY), obliczenia procentowe, konwersja typów danych

## Kluczowe wnioski
1. Obliczono współczynnik konwersji (Click-to-Sale).
2. Zidentyfikowano, że model Toyota ma wyższą konwersję niż Mazda.
3. Wykorzystano złączenia (LEFT JOIN), aby nie utracić danych o klientach, którzy nie dokonali zakupu.
